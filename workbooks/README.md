# Workbooks

Place the master **Lithuania retention / fiscal scenario** Excel files here (`.xlsx`).

Default filenames used by automation:

- `final_lithuania_model (1).xlsx`
- `final_lithuania_model (2).xlsx`

Scripts also accept any path via CLI. If you add a new version, either keep a matching name or pass `--workbook` / positional args explicitly.

Expected structure inside each workbook: **Inputs**, **Scenario_Drivers**, **Model**, **Summary**, **Notes** (see root **README.md**).
